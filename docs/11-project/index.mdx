## カウンター (基本編)

数を数えることができるアプリです。

### ルール

- 数を表示する部分があります。
- `+1` と書かれたボタンをクリックすると、数が 1 ずつ増えます。

<video src={counterVideo} controls />

### ヒント

```html
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="utf-8" />
    <title>カウンター</title>
  </head>
  <body>
    <!-- 数を表示する部分をここに書く -->
    <!-- ボタンをここに書く -->
  </body>
</html>
```

```js
let counter = 0;

function plus() {
  // ボタンがクリックされたときの処理を書く
}

// ボタンがクリックされたときに plus 関数を呼び出す
document.getElementById(/* ボタンのid */).onclick = plus;
```

### 解答例

<Answer title="カウンター (基本編)">

```html
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="utf-8" />
    <title>カウンター</title>
  </head>
  <body>

    <button id="plus" type="button">+1</button>
  </body>

```

## カウンター (応用編)

### ルール

- 9 まで数えたら、次は 10 ではなく 0 に戻るようにしましょう。

<video src={counterResetVideo} controls />

### ヒント

```js
let counter = 0;

function plus() {
  if (/* 何らかの条件 */) {
    // カウンターを 1 増やす処理
  } else {
    // カウンターを 0 に戻す処理
  }
  document.getElementById("counter").textContent = counter;
}

document.getElementById("plus").onclick = plus;
```

### 解答例

<Answer title="カウンター (応用編)">

```js
let counter = 0;

function plus() {
  if (counter < 9) {
    counter = counter + 1;
  } else {
    counter = 0;
  }
  document.getElementById("counter").textContent = counter;
}

document.getElementById("plus").onclick = plus;
```

## おみくじ

今日の運勢をランダムで表示するアプリです。

### ルール

- 20%の確率で `大吉` と表示し、文字色を赤色にします
- 50%の確率で `吉` と表示し、文字色を黒色にします
- 30%の確率で `凶` と表示し、文字色を青色にします

<video src={omikujiVideo} controls />

### ヒント

:::info ランダムな数を使う

`Math.random` <Term>関数</Term>は、呼び出すたびに変わる 0 以上 1 未満の乱数 (ランダムな数) を<Term>戻り値</Term>として<Term>返し</Term>ます。

```javascript
document.write(Math.random()); // 0.8924949384481442
```

発生させた乱数を `r` などの名前の<Term>変数</Term>に保存して、確率の計算に使いましょう。

例えば `r` が 0 以上 0.2 未満になる確率は 20% になりますね。同様に、`r` が 0.2 以上 0.7 未満になる確率は 50%、`r` が 0.7 以上 1 未満になる確率は 30% です。

```javascript
let r = Math.random();
if (r < 0.2) {
  // 20% の確率で実行される処理を書く
} else if (/* 何らかの条件 */) {
  // 50% の確率で実行される処理を書く
} else {
  // 30% の確率で実行される処理を書く
}
```

:::

```html
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="utf-8" />
    <title>おみくじ</title>
  </head>
  <body>
    <!-- 結果を表示する部分をここに書く -->
    <!-- ボタンをここに書く -->
  </body>
</html>
```

```js
function omikuji() {
  let r = Math.random();
  if (r < 0.2) {
    // 20% の確率で実行される処理を書く
  } else if (/* 何らかの条件 */) {
    // 50% の確率で実行される処理を書く
  } else {
    // 30% の確率で実行される処理を書く
  }
}

document.getElementById(/* おみくじを引くボタンのid */).onclick = omikuji;
```

### 解答例

<Answer title="おみくじ">

```html
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="utf-8" />
    <title>おみくじ</title>
  </head>
  <body>
    <p id="result">ここに結果が表示されます</p>
    <button id="omikuji-button" type="button">おみくじを引く</button>
  </body>
</html>
```

```js
function omikuji() {
  let r = Math.random();
  if (r < 0.2) {
    document.getElementById("result").textContent = "大吉";
    document.getElementById("result").style.color = "red";
  } else if (r < 0.7) {
    document.getElementById("result").textContent = "吉";
    document.getElementById("result").style.color = "black";
  } else {
    document.getElementById("result").textContent = "凶";
    document.getElementById("result").style.color = "blue";
  }
}

document.getElementById("omikuji-button").onclick = omikuji;
```
