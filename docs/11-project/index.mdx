## カウンター

数を数えることができるアプリです。

### ルール

- 数を表示する部分があります。
- `+1` と書かれたボタンをクリックすると、数が 1 ずつ増えます。

<video src={counterVideo} controls />

### ヒント

```html
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="utf-8" />
    <title>カウンター</title>
  </head>
  <body>
    <!-- 数を表示する部分をここに書く -->
    <!-- ボタンをここに書く -->
  </body>
</html>
```

```js
let counter = 0;

function plus() {
  // ボタンがクリックされたときの処理を書く
}

// ボタンがクリックされたときに plus 関数を呼び出す
document.getElementById(/* ボタンのid */).onclick = plus;
```

### 解答例

<Answer title="カウンター">

```html
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="utf-8" />
    <title>カウンター</title>
  </head>
  <body>
    <p id="counter">0</p>
    <button id="plus" type="button">+1</button>
  </body>
</html>
```

```js
let counter = 0;

function plus() {
  counter = counter + 1;
  document.getElementById("counter").textContent = counter;
}

document.getElementById("plus").onclick = plus;
```

## おみくじ

今日の運勢をランダムで表示するアプリです。

### ルール

- 20%の確率で `大吉` と表示し、文字色を赤色にします
- 50%の確率で `吉` と表示し、文字色を黒色にします
- 30%の確率で `凶` と表示し、文字色を青色にします

<video src={omikujiVideo} controls />

### ヒント

- `Math.random` <Term>関数</Term>は、呼び出すたびに変わる 0 以上 1 未満の乱数 (ランダムな数) を<Term>返し</Term>ます。

```javascript
document.write(Math.random()); // 0.8924949384481442
```

- 発生させた乱数を `r` などの名前の<Term>変数</Term>に保存して確率の計算に使いましょう。すると、例えば `r` が 0 以上 0.2 未満になる確率は 20% になりますね。

```javascript
let r = Math.random();
if (r < 0.2) {
  // 20%の確率で実行される
} else if (条件式2) {
  // 50%の確率で実行される
} else {
  // 30%の確率で実行される
}
```

- クリックするためのボタンと、テキストを表示して色を変えるための `p` <Term>要素</Term>の両方を <Term>JavaScript</Term> から取得してみましょう。

```html
<button id="omikuji-button" type="button">おみくじを引く</button>
<p id="result"></p>
```

```javascript
let omikujiButton = document.getElementById("omikuji-button");
let result = document.getElementById("result");

function omikuji() {
  result.textContent = "結果";
}
omikujiButton.onclick = omikuji;
```
